# Node Js Learning Repo
## Chapter 1
This Chapter is to learn basic backend knowlegde like CRUD in **Node Js** + **Express Js**.
> this Chapter is only learn basic backend using NodeJs without frontend

## Chapter 2
This chapter is to develop a to-do list application using backend knowledge.

Simple frontend is used to show the to-do list.

**Tech Stack**
- NodeJs + ExpressJs
- JWT Authentication
- NoSQL
- Middleware
- NPM (Nodeman)

## Chapter 3
This chapter is to migrate the to-do list into differet technologies.
> Same application but different technologies are used

**Tech Stack**
- Migrate from `NoSQL` to `PostgreSQL`
- Dockerise the application using `Docker`
- `yaml` file used for set up configuration in `Docker`
- `Prisma ORM` used for high productivity
